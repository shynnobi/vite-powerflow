# IMPORTANT - Storybook Reference Branch

## About this branch

This branch (`reference/storybook-advanced-features`) contains advanced Storybook features that are not yet integrated into the main branch but will serve as a reference for future implementations.

## Why keep this branch?

- **Technical reference**: Contains advanced examples and configurations for Storybook
- **Work in progress**: Features under development intended for later integration
- **Living documentation**: Demonstrates design approaches we want to incorporate

## How to use this branch

Do not delete this branch when cleaning up repository branches. It is intentionally kept separate from `main` and `dev` to preserve these reference examples.

```bash
# To view the content
git checkout reference/storybook-advanced-features

# To see differences from main
git diff main..reference/storybook-advanced-features
```
