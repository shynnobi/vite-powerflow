## üöÄ Quick Start

```bash
# Using the official CLI (recommended)
npx create-powerflow-app my-project

# Using degit
npx degit shynnobi/vite-powerflow my-project

# OR clone the repository
git clone https://github.com/shynnobi/vite-powerflow my-project

cd my-project
pnpm install
pnpm dev
```
