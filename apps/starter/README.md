## Quick Start

Choose your preferred development approach:

### 🐳 Docker

1. Open your app in your favorite IDE

2. `Reopen in Container` when prompted (Dev Container)
   - The first time, wait for the installation (can take a few minutes).
   - On subsequent launches, you'll connect directly to the container.

3. Launch dev server:

   ```bash
   pnpm dev
   ```

### 💻 Local Development

1. Ensure you have **Node.js** and **pnpm** installed:

   ```bash
   # Install pnpm if needed
   npm install -g pnpm
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Launch dev server:

   ```bash
   pnpm dev
   ```

4. Start developing!
